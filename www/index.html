Download Addon Learn More

Everything You Need in One Addon

BiS gear lists, real-time loot tracking, upgrade detection, and more — all without alt-tabbing.

🧩

BiS Gear Grid

Visual icon grid showing Best in Slot gear for every class and spec. Green border = equipped, red = you need it. Items you already have are dimmed so you focus on upgrades.

📦

Loot Tracker

Real-time loot tracking in dungeons and raids. See every drop organized by boss, with item level, gear track (Hero 3/6, Myth 2/6), and who received each item.

⬆

Upgrade Detection

Instantly see which drops are upgrades for you with a green arrow overlay. Item levels are highlighted in green when they beat your currently equipped gear.

⭐

Wishlist

Right-click any loot item to bookmark it. Track which items you want to roll for at the end of the raid. View your full wishlist with a single command.

🔊

Sound Alerts

Hear a distinct sound when Epic or Legendary items drop. Never miss an important piece of loot, even when you're focused on the fight.

💎

Enchants, Gems & Consumables

Complete BiS lists for enchants, gems, and consumables in the same icon grid format. Green = you have it, red = you're missing it.

📋

Talent Builds

Recommended talent builds for Raid, M+, Delves and PvP. One-click copy of import strings directly into WoW's talent UI.

🌐

Dual Source

BiS data from both Icy Veins and Wowhead. Switch between sources in-game to compare recommendations. Supports all 13 classes and 39 specs.

See It in Action

Commands Reference

Everything is accessible via the minimap button or slash commands.

CommandDescription
/adhd bisToggle the BiS gear panel
/adhd lootToggle the Loot Tracker window
/adhd loot newStart a new loot session (auto-names with instance)
/adhd loot new My RaidStart a new loot session with a custom name
/adhd loot startResume loot tracking
/adhd loot stopPause loot tracking
/adhd loot summaryPrint full session summary to chat
/adhd loot wishlistShow all wishlisted items
/adhd loot helpShow all loot tracker commands

Grid Controls

Shift+Click — Preview item (Dressing Room)  •  Ctrl+Click — Link item to chat  •  Right-Click — Toggle wishlist bookmark

How to Install

Up and running in under 2 minutes.

1

Download the Addon

Download the ADHDBiS addon zip and extract it into your World of Warcraft/_midnight_/Interface/AddOns/ folder.

2

Run the Companion App

Download and run the ADHDBiS Updater for your platform. Select your source (Icy Veins or Wowhead), pick your classes, and click Update.

3

Reload & Play

Type /reload in WoW, then /adhd bis to open the BiS panel. The minimap button gives you quick access to everything.
